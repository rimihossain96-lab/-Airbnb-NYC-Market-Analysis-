# -Airbnb-NYC-Market-Analysis-
Exploratory Data Analysis (EDA) of Airbnb market data to identify pricing trends, customer behavior, and demand patterns. The project uses Python, Pandas, Matplotlib, and Seaborn to clean data, visualize relationships between price, room type, availability, and reviews, and generate insights to understand factors influencing listing performance.

## 📌 Project Overview  

This project analyzes Airbnb listing data in New York City to uncover patterns, pricing trends, customer behavior, and neighborhood-level demand insights.  

The dataset used: *Airbnb_NYC2019.csv*  

The analysis is performed using Python in Jupyter Notebook (Airbnb.ipynb).  

---

## 🎯 Objectives  

- Analyze Airbnb pricing trends  
- Study room type distribution  
- Explore relationship between reviews and price  
- Identify neighborhood-based demand patterns  
- Generate meaningful market insights

## 📊 Exploratory Data Analysis (EDA)  

The following analysis was performed:

- Data Cleaning (Handling null & infinite values)  
- Univariate Analysis  
- Bivariate Analysis  
- Price vs Room Type (Boxplot)  
- Reviews vs Price (Scatterplot)  
- Neighborhood-wise Analysis  
- Availability Analysis  
- Correlation Heatmap  

---

## 📈 Key Insights  

- Entire homes/apartments generally have higher prices.  
- Moderately priced listings tend to receive more reviews.  
- Manhattan listings show higher average pricing.  
- Price distribution contains significant outliers.  
- Availability varies significantly across neighborhoods.
